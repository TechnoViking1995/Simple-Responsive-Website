## Project Overview
This project is a responsive website built using Bootstrap.css as the base styling template. The HTML and JavaScript have been extensively customized to match the project requirements.

## Customizations
- HTML layout improvements for better structure and content flow.
- JavaScript interactivity added to enhance user experience.
- Bootstrap.css was used as a template, with no further CSS modifications.
